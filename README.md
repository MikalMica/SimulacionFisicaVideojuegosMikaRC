\## MANUAL DE USUARIO



Simulador espacial en el que el único objetivo es interactuar con las cosas y ver como reacciona el entorno a eventos que inicie el jugador



Controles:



h → Gira la nave a la izquierda

k → Gira la nave a la derecha

j → Hace avanzar a la nave

z → Dispara la nave (sólido)

x → Disparo desde la cámara (partícula)

1 → Desactiva la gravedad de todo

2 → Hace explotar todos los planetas que puedan hacerlo (también es posible hacerlos explotar disparándoles con la nave)

3 → Desactiva el tornado que hace que las partículas de ringed planet giren

4 → Mata al alien :(

5 → Hace que la estrella del sistema solar se convierta en supernova y se genere un agujero negro
